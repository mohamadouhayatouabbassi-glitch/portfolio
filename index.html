Personal Project
Credit Card Fraud Detection Imbalanced Classification

Tackled a highly skewed real-world dataset of 284,807 credit card transactions (0.17% fraud rate) using XGBoost with stratified splitting and scale_pos_weight. Prioritised PR-AUC over ROC-AUC to handle class imbalance. Calibrated probability thresholds to minimise false alarms and serialised the final model for pipeline integration.

80% Recall 90.5% Precision +38% vs Baseline +21.7% PR-AUC
